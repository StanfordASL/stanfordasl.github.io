# ASL Website

Built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com).

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```
